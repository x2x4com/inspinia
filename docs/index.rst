Flask Template Project
======================


Template flask application using Flask-Admin, Flask-Security, SQLAlchemy, Sentry, and Celery.

Setup
-----

If you have `pip <https://pip.pypa.io/en/latest/installing.html>`__ you can install by executing the following command:

.. code:: bash

    pip install flask_template

Or to manually install, execute the following commands:

.. code:: bash

    git clone https://github.com/20miler10/flask-template
    cd flask_template/
    python setup.py install
